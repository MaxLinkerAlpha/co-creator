## 📖 项目简介

**Co-creator** 是一款面向多语言写作者的生产力工具。

**核心特性**:
- 🌐 **多语言并发翻译** - 支持英/日/法/德/俄/阿/拉丁等
- ⚡ **双触发同步机制** - 段落完成立即触发 + 停止打字防抖触发
- 📝 **智能标题同步** - Markdown 标题（#）识别已翻译内容，不再重复调用 API
- 🔄 **双向同步** - 源文本 → 目标文本；目标文本 → 源文本（逆向反推）
- 🌍 **20位AI助教** - 10男10女，老青少三代，不同国家文化背景
- 🎨 **4 种主题** - Apple 极简 / 暗黑模式 / 护眼模式 / 罗马古典
- 🛡️ **XSS 防护** - 全站转义，防御注入攻击
- ⚡ **并发安全** - AbortController 防止竞态条件

### 设计哲学

**零异步复杂度**：所有数据直接内嵌在 `data.js` 中，无需 fetch、无需等待、无竞态风险。

```javascript
// 100% 同步执行
TutorSystem.switch('marcus');  // 立即切换，无延迟
```

### 命名空间

| 命名空间 | 职责 |
|---------|------|
| `Utils` | 工具函数（防抖、XSS转义、Markdown检测） |
| `Store` | localStorage 封装 |
| `API` | API 通信（指数退避重试 2 次） |
| `SyncEngine` | 区块级同步引擎（AbortController 竞态防护） |
| `TutorSystem` | 20位助教系统（5种吐槽模式） |
| `UI` | 视图更新与主题管理 |

## 📝 使用说明

### 基本操作

1. **选择助教**：点击右下角头像 → 下拉菜单选择
2. **开始写作**：在左侧输入中文，右侧自动同步翻译
3. **Markdown 标题**：以 `#` 开头的行智能处理
   - 内容未变仅加 `#` → 目标列直接加 `#`，**不调用 API**
   - 新标题内容 → 翻译后自动加 `#`

### 翻译触发机制

| 触发方式 | 机制 | 延迟 |
|---------|------|------|
| **段落完成** | 检测到 `\n\n`（两次回车） | 立即 |
| **停止打字** | 防抖机制 | 1500ms |

两种机制**并行工作**，确保实时性与稳定性。

### AI 助教吐槽模式

| 模式 | 说明 | 触发条件 |
|------|------|----------|
| **[高频]** 紧盯你写每一个字 | 几乎实时吐槽 | 每10字符，冷却1分钟 |
| **[正常]** 段落结束吐槽 | 检测到段落完成 | 每50字符或段落结束，冷却3分钟 |
| **[低频]** 忍无可忍才说 | 高阈值触发 | 每200字符，冷却5分钟 |
| **[慢写]** 吐槽你写得慢 | 长时间无输入 | 60秒无输入触发 |
| **[手动]** 只有点击才吐槽 | 不自动触发 | 点击按钮 |

**吐槽形式**：
- **自动模式**：气泡弹窗（20-35字），节省 Token
- **手动模式**：侧边面板完整格式（问候+点评+教学）

## ⚙️ 技术栈

- **纯原生** HTML/CSS/JS，无构建工具
- **marked.js** - Markdown 渲染（CDN）
- **Google Fonts** - Cinzel, Noto Serif SC

### CSS 架构

**8pt 网格系统**
```css
--space-1: 4px;  --space-5: 24px;
--space-2: 8px;  --space-6: 32px;
--space-3: 12px; --space-4: 16px;
```

**DRY 主题系统**
```css
[data-tutor="theme-german"] {
  --tutor-primary: #cc0000;
  --tutor-secondary: #ffd700;
  /* 自动应用到所有组件 */
}
```

### 安全机制

**XSS 防护**：所有 `innerHTML` 操作前经过 `Utils.escapeHTML()` 转义。

**竞态防护**：使用 `AbortController` 取消过期的翻译请求，确保结果显示一致性。

**指数退避**：API 调用失败时自动重试 2 次，延迟分别为 1s、2s。
