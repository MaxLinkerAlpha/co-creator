## 🎯 极简使用指南（小白专用）

### 🌐 方式一：在线使用（推荐）

**直接访问在线版本，无需下载安装！**

> 🚀 访问地址：**[https://maxlinkeralpha.github.io/co-creator/](https://maxlinkeralpha.github.io/co-creator/)**

- 首次使用需要配置 **API Key**
- 推荐 [SiliconFlow](https://cloud.siliconflow.cn)，注册即送免费额度
- 配置后即可使用 Qwen 等模型进行翻译
- 数据保存在浏览器本地，刷新不丢失

### 💻 方式二：本地运行

### 第一步：下载项目

1. 点击页面右上角的绿色按钮 **「Code」**
2. 选择 **「Download ZIP」**
3. 解压下载的文件到任意文件夹

### 第二步：启动程序

**Windows 用户**：
1. 打开解压后的文件夹
2. 在文件夹地址栏输入 `cmd` 然后按回车
3. 在弹出的黑色窗口中输入：`python -m http.server 8001`
4. 打开浏览器，访问 `http://localhost:8001`

**Mac 用户**：
1. 打开「终端」应用
2. 输入 `cd `（注意有空格），然后把解压的文件夹拖进终端窗口
3. 按回车，然后输入：`python3 -m http.server 8001`
4. 打开浏览器，访问 `http://localhost:8001`

### 第三步：开始使用

1. 页面打开后，**直接开始写作**即可
2. 默认使用**免费模型**，无需配置 API 密钥
3. 右侧会自动翻译成英语
4. 点击 **「➕ 目标栏」** 可以添加更多语言
5. 右下角的助教头像可以点击，查看助教吐槽

### 常见问题

**Q: 页面打不开？**
- 确保终端窗口没有关闭
- 确保地址输入正确（注意是 `localhost` 不是 `locallost`）

**Q: 翻译没反应？**
- 免费模型可能有延迟，请耐心等待
- 尝试输入更多文字后再等待

**Q: 想用更好的模型？**
- 点击工具栏的「🤖 模型」下拉菜单
- 选择其他模型（需要配置 API 密钥）

## ✨ 项目简介

**Co-creator** 是一款面向多语言写作者的生产力工具。写作时，AI助教会随机出没，用各国语言和文化背景吐槽你的文字。

**核心特性**:
- 🎲 **随机助教模式** - 默认启用，20位助教随机出没吐槽
- 🌐 **多语言并发翻译** - 支持 8 种语言（英/日/西/法/德/俄/阿/拉丁）
- ⚡ **双触发同步** - 段落完成立即触发 + 停止打字自动触发
- 🔄 **双向同步** - 源文本→目标文本；目标文本→源文本
- 🎭 **沉浸式预览** - 停止输入3秒后自动渲染Markdown
- 📚 **动态术语库** - 只匹配文本中实际出现的术语
- 🎨 **4种主题** - Apple极简 / 暗黑模式 / 护眼模式 / 罗马古典

## 📝 使用说明

### 基本操作

1. **开始写作**：在左侧输入文字，右侧自动翻译
2. **添加目标栏**：点击「添加并发目标栏」同时翻译多种语言
3. **选择助教**：点击右下角头像 → 下拉菜单选择（或保持随机模式）

### 翻译触发机制

| 触发方式 | 说明 |
|---------|------|
| **段落完成** | 输入两个回车（`\n\n`）立即触发 |
| **停止打字** | 停止输入 1.5 秒后自动触发 |

### 🎭 沉浸式自动预览

开启「沉浸式自动预览」后：
- 停止输入 3 秒 → 自动切换到 Markdown 渲染预览
- 任意键盘输入 → 立即返回编辑模式

### 🎯 Focus is Truth 逆向同步

当你在**目标语言文本框**中直接编辑时，系统会自动将内容翻译回源语言。

**使用场景**：
- 直接润色翻译结果
- 从其他文档粘贴目标语言内容

### AI 助教吐槽模式

| 模式 | 说明 |
|------|------|
| 👀 **高频** | 紧盯你写每一个字 |
| 📝 **正常** | 段落结束时吐槽（推荐） |
| 😤 **低频** | 忍无可忍才说 |
| 🐢 **慢写** | 吐槽你写得慢 |
| 🖱️ **手动** | 只有点击才吐槽 |

### 📚 动态术语库

导入 CSV 术语文件后，系统只会将**实际出现**的术语加入翻译提示词，节省 Token。

```csv
源术语,目标术语
人工智能,Artificial Intelligence
区块链,Blockchain
```

## ⚙️ 配置说明

### config.js

```javascript
const CONFIG = {
    API_KEY: 'sk-your-api-key-here',        // API 密钥
    API_URL: 'https://api.xxx.com/v1/chat/completions',  // API 端点
    MODEL_NAME: 'Qwen/Qwen3.5-9B',          // 默认模型
    TEMPERATURE: 0.2                         // 翻译温度
};
```

### 可选模型

| 模型 | 特点 |
|------|------|
| ⚡ Qwen3.5-4B | 最快响应 |
| ⚖️ Qwen3.5-9B | 平衡之选（推荐） |
| 🎯 Qwen3.5-35B | 专业级 |
